# Docs

This directory is the canonical human-readable architecture layer for `cai-platform-v2`.

## Structure
- [architecture/](architecture/README.md): platform boundaries, dependency rules, and topology notes
- [contracts/](contracts/README.md): human-readable explanations of shared platform contracts
- [backends/](backends/README.md): backend-specific guidance and conformance notes
- [apps/](apps/README.md): app-level docs for the platform API and CAI orchestration app
- [adr/](adr/README.md): architecture decision records
- [operations/](operations/README.md): local development and operational runbooks

## Rule
- Docs are the canonical human-readable architecture layer.
- Package READMEs should summarize local boundaries, but they do not replace `docs/`.
